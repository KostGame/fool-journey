# Путь Шута

Статический браузерный прототип про путь старших арканов Таро от Шута к Миру. Между большими главами появляются короткие дорожные события младших арканов.

## Статус

Текущая линия разработки: `SU-008` - расширение диалогового пути до блока `Иерофант → Влюблённые → Колесница → Сила → Отшельник → Колесо Фортуны → Справедливость` с первым слоем карт-инструментов Шута.

## Что уже есть

- Vite + TypeScript + Vitest.
- Главный экран на русском языке.
- Первый играбельный путь Шута.
- Диалоговый vertical slice для режима `Путь Шута` с major scenes, minor events и helper appearances.
- Расширенный диалоговый блок старших арканов до Справедливости.
- Данные для 22 старших арканов и всех 56 младших арканов.
- Короткие дорожные события между большими главами.
- Минимальная механика инвентаря карт: `earnedCards`, `inventoryCards`, `knownCards`, а также метки `Получено`, `Применено`, `Помощник`.
- Сохранение прогресса в `localStorage`.
- Placeholder-режимы для будущих экранов.
- GitHub Actions CI и GitHub Pages.
- Сконструирован первый playable route: маршрут, result screens, reset/replay и local smoke-test Pages.

## Структура

- `src/data` - карты, главы, encounters и minor events.
- `src/domain` - модель состояния, трактовки, прогресс и storage.
- `src/app.ts` - рендер интерфейса и обработка действий.
- `src/style.css` - мобильная визуальная оболочка.
- `docs` - проектная документация.
- `tasks/SU` - карточки крупных задач.
- `reports` - отчёты Codex по PR.
- `.github/workflows` - CI и Pages.

## Процесс работы

1. Задача описывается в GitHub Issue.
2. Для крупной задачи создаётся файл в `tasks/SU/`.
3. Работа ведётся в feature-ветке.
4. Если задача готова к проверке, Codex открывает обычный Open PR.
5. Пользователь вручную проверяет визуальные изменения.
6. Отчёт кладётся в `reports/`.
7. PR не мержится без ручной проверки и явного решения по merge.

## SU-007

Текущая линия развития: `home → scene → result → next scene`.

- Главный экран больше не содержит длинную активную сцену.
- `Продолжить историю` открывает отдельный экран сцены.
- После выбора открывается отдельный экран результата.
- `Дальше` переносит игрока к следующей сцене сверху.

## SU-008

- В dialogue format добавлены Иерофант, Влюблённые, Колесница, Сила, Отшельник, Колесо Фортуны и Справедливость.
- В выборах появились действия с условиями по ранее полученным картам.
- После выбора UI показывает короткие статусы `Получено`, `Применено` и `Помощник`.
- Инвентарь карт сохраняется и восстанавливается через `localStorage`.
## Дневник Шута

В PR-010 добавлен отдельный secondary screen `Дневник Шута`.

- Главный CTA остаётся `Продолжить историю`.
- Дневник открывается отдельным экраном из home и из completion.
- Дневник показывает полученные карты, применённые карты, помощников и главы пути.
- Экран собирается из существующего inventory state и не раскрывает весь справочник из 78 карт.

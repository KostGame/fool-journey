# Путь Шута

Статический браузерный прототип игры для изучения Таро через короткие сюжетные сцены, трактовки и мягкие выборы.

## Статус

PR-000: bootstrap.

## Что уже умеет прототип

- Открывается на Vite + TypeScript.
- Показывает главный экран на русском языке.
- Содержит первый кликабельный цикл Шута.
- Сохраняет прогресс в `localStorage`.
- Работает без backend, авторизации и внешних API.

## Команды

- `npm install`
- `npm run dev`
- `npm run check`
- `npm test`
- `npm run build`
- `npm run preview`

## Структура

- `src/data` - стартовые данные по картам, главам и ситуациям.
- `src/domain` - доменная модель, трактовки, прогресс и storage.
- `src/app.ts` - рендер и обработка действий интерфейса.
- `src/style.css` - мобильная визуальная оболочка.
- `docs` - проектная документация.
- `tasks/SU` - карточки крупных задач.
- `reports` - отчёты по PR.
- `.github/workflows` - CI и Pages.

## Процесс работы

1. Задача описывается в GitHub Issue.
2. Крупная задача получает файл в `tasks/SU/`.
3. Код делается в feature-ветке.
4. Codex открывает Draft PR.
5. После проверки пользователь смотрит визуальные изменения вручную.
6. Отчёт кладётся в `reports/`.
7. Без ручной проверки PR не переводится в merge-ready.

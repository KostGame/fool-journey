# SU-010: Add Fool Journal and card collection

## Цель

Добавить отдельный экран `Дневник Шута`, где игрок может увидеть след путешествия: полученные карты, применённые карты, помощников и главы пути.

## Scope

- Отдельный secondary screen для дневника.
- Использование существующего `inventory state`.
- Краткая сводка прогресса.
- Секции:
  - полученные карты;
  - применённые карты;
  - помощники;
  - главы пути.
- Пустые состояния для новых сохранений.
- Сохранение совместимости со старыми сейвами.

## Не делать

- Не превращать дневник в справочник всех 78 карт.
- Не добавлять backend, авторизацию, внешние API или RPG-экономику.
- Не ломать основной story flow.

## Проверки

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## Риски

- Слишком длинный экран на мобильной ширине.
- Дублирование данных между inventory и дневником.
- Потеря совместимости со старыми сохранениями.

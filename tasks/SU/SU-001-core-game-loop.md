# SU-001 — Core game loop for Fool, Magician and High Priestess

## Контекст

После bootstrap-итерации проект уже умеет показывать главный экран, хранить локальный прогресс и запускать стартовую сцену Шута. Следующий шаг — собрать короткий обучающий цикл вокруг трёх аркан: Шута, Мага и Жрицы.

## Цель

Сделать законченный учебный эпизод `Шут → Маг → Жрица`, где игрок проходит три сцены подряд, получает мягкую обратную связь, накапливает XP, видит прогресс эпизода и финальный экран завершения.

## Не делаем

- Не добавляем backend, авторизацию, базу данных и внешние API.
- Не делаем сложные расклады.
- Не реализуем все 78 карт.
- Не используем Draft PR.
- Не мержим PR самостоятельно.
- Не делаем перевёрнутые карты в этой задаче.

## Состав работ

- Расширить сюжетный режим «Путь Шута» до трёх последовательных сцен.
- Добавить мягкие формулировки обратной связи после выбора.
- Сохранять прогресс, XP, завершённые сцены и последнюю обратную связь в `localStorage`.
- Показать на главном экране уровень, XP, карту текущей главы и прогресс эпизода `1 / 3`, `2 / 3`, `3 / 3`.
- Добавить экран завершения эпизода с кратким итогом.
- Обновить доменную модель и тесты.
- Обновить документацию и отчёт.

## Критерии готовности

1. Есть последовательность `Шут → Маг → Жрица`.
2. У каждой сцены есть сюжет, вопрос, 2-3 варианта, обратная связь и XP.
3. На главном экране виден прогресс эпизода.
4. После Жрицы показывается экран завершения.
5. Состояние сохраняется и восстанавливается.
6. Placeholder-режимы продолжают работать.
7. Проверки проходят.
8. PR открыт как обычный Open PR.

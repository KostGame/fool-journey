# SU-009 — Complete dialogue arc from Hanged Man to World

## Контекст

После SU-008 диалоговый путь Шута уже доведён до `Справедливости`, а текущая задача должна закрыть финальный блок старших арканов и довести основную историю до `Мира` без возврата к длинной странице.

## Цель

Перевести финальный блок `Повешенный → Мир` в dialogue/text quest формат, сохранить screen-by-screen flow и активно использовать минимальный inventory layer Шута.

## Что нужно сделать

- Добавить dialogue-сцены для `Повешенного`, `Смерти`, `Умеренности`, `Дьявола`, `Башни`, `Звезды`, `Луны`, `Солнца`, `Суда` и `Мира`.
- Сохранить мобильный flow `home → scene → result → next scene → completion`.
- Использовать карты как инвентарь Шута:
  - добавить новые inventory rewards;
  - добавить выборы, которые применяют ранее полученные карты;
  - показать короткие статусы `Получено`, `Применено`, `Помощник`.
- Добавить минимум 4 новых appearances помощников из двора.
- Усилить финальный экран после `Мира`.
- Исправить кодировку и убедиться, что UI-строки отображаются нормально.

## Ограничения

- Не добавлять новые режимы.
- Не делать сложную RPG-экономику.
- Не возвращать длинную страницу.
- Не переписывать весь проект заново.

## Критерии готовности

- Все указанные major arcana доступны в dialogue format.
- Inventory-связки работают и сохраняются в `localStorage`.
- Проверки `npm run check`, `npm test`, `npm run build`, `git diff --check` проходят.
- Подготовлен обычный Open PR, не Draft.
- Есть отчёт в `reports/SU-009-complete-dialogue-arc.md`.

# SU-000 — Bootstrap «Путь Шута»

## Первая версия ТЗ

Это первый вариант технического задания после создания рабочих каталогов проекта.

## Созданные каталоги

- `.github/workflows`
- `docs`
- `reports`
- `src/data`
- `src/domain`
- `tasks/SU`

## Цель

Собрать минимальный статический прототип браузерной игры «Путь Шута» на Vite + TypeScript + Vitest.

## Скоуп

- Главный экран.
- Первый игровой цикл.
- Сохранение прогресса в `localStorage`.
- Placeholder-экраны для остальных режимов.
- CI и GitHub Pages.
- Документация и отчёт.

## Не делаем

- Не делаем 78 карт.
- Не добавляем backend.
- Не используем внешние API.
- Не добавляем авторизацию.
- Не строим сложную систему раскладов.

## Проверки

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## Риски

- Ошибка в кодировке русских строк.
- Случайная горизонтальная прокрутка на мобильной ширине.
- Несовпадение путей для GitHub Pages.

# tasks/SU

Здесь лежат крупные задачи уровня SU.

## Правила

- Каждая большая задача получает отдельный markdown-файл.
- Задачи описываются до начала реализации.
- В файле фиксируются цель, скоуп, проверки и риски.
- Для PR-000 используется `SU-000-bootstrap.md`.
- Для SU-010 используется `SU-010-fool-journal-card-collection.md`.

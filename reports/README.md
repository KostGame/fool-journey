# reports

Здесь лежат финальные отчёты Codex по PR.

## Правила

- один отчёт на один PR;
- отчёт пишется на русском языке;
- отчёт честно фиксирует ограничения, проверки и ручную валидацию;
- имя файла совпадает с задачей, например `PR-000-bootstrap.md` или `SU-012-tarot-card-thumbnails.md`.
- для SU-017 используется `SU-017-result-feedback-card-rewards.md`.

# CHECKS

## Обязательные проверки

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## Локальная разработка

- `npm run dev`
- `npm run preview`

## Ручная проверка на смартфонной ширине

- Нет горизонтальной прокрутки.
- Кнопки удобно нажимаются одной рукой.
- Главный экран выглядит чисто и не ломается.
- Цикл пути показывает и major, и minor шаги.
- Прогресс сохраняется после перезагрузки.
- Сброс прогресса работает предсказуемо.
- Путь завершается на карте Мира и может быть пройден повторно.
- Placeholder-режимы не ломают навигацию.

## Перед PR

Сначала проверки, потом отчёт, потом обычный Open PR.

## SU-005 Mobile Hotfix

- На ширине 360–430 px блок режимов не имеет горизонтальной прокрутки.
- Карточки режимов не сжимаются в узкие колонки и не налезают друг на друга.
- Названия и описания режимов читаются и переносятся нормально.


## SU-004 Pages smoke-test

- Открыть Pages URL на ширине смартфона.
- Проверить, что главный экран показывает прогресс, маршрут и список режимов без горизонтальной прокрутки.
- Пройти несколько major сцен, дождаться minor события, посмотреть result screen и вернуться к journey.
- Дойти до completion, проверить кнопку `Повторить историю`.
- Нажать reset, убедиться, что появляется подтверждение и прогресс сбрасывается.
- Перезагрузить страницу и убедиться, что `localStorage` восстанавливает состояние.
- Проверить перезапуск после completion и отсутствие поломок placeholder modes.

## SU-006 Dialogue Quest smoke-test

- Открыть `Путь Шута` и пройти vertical slice на мобильной ширине 360–430 px.

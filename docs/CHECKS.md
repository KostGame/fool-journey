# CHECKS

## Обязательные проверки

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## Локальная разработка

- `npm run dev`
- `npm run preview`

## Ручная проверка на смартфонной ширине

- Нет горизонтальной прокрутки.
- Кнопки удобно нажимаются одной рукой.
- Главный экран выглядит чисто и не ломается.
- Цикл пути показывает и major, и minor шаги.
- Прогресс сохраняется после перезагрузки.
- Сброс прогресса работает предсказуемо.
- Путь завершается на карте Мира и может быть пройден повторно.
- Placeholder-режимы не ломают навигацию.

## Перед PR

Сначала проверки, потом отчёт, потом обычный Open PR.

# CHECKS

## Обязательные проверки

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## Локальная разработка

- `npm run dev`
- `npm run preview`

## Ручная проверка на смартфоне

- Нет горизонтальной прокрутки.
- Кнопки читаются и нажимаются одной рукой.
- Главный экран открывается без артефактов.
- Прогресс пути корректно показывает `N / 22`.
- Прогресс сохраняется после перезагрузки.
- Сброс прогресса работает предсказуемо.
- Полный путь завершается на карте `Мир`.
- После завершения путь можно пройти повторно.
- Placeholder-экраны не ломают навигацию.

## Перед PR

Сначала проверки, потом отчёт, потом обычный Open PR.

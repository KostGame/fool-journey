# DECISIONS

## D-001

Проект остаётся статическим фронтендом без backend и без внешних API.

## D-002

Прогресс хранится только в `localStorage`.

## D-003

В PR-000 используются только три карты: Шут, Маг и Жрица.

## D-004

Трактовка собирается композиционно: карта, контекст, ориентация и точечные overrides.

## D-005

Деплой идёт через GitHub Pages с базовым путём `/fool-journey/`.

## D-006

В интерфейсе не используется слово «шпаргалка».

## D-007

Первая реализация делается mobile-first, с приоритетом смартфона.

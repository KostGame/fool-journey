# PROJECT_BRIEF

## Версия 0.1

Это первая версия технического задания для проекта «Путь Шута». Она зафиксирована после создания базовых каталогов и первого каркаса приложения.

## Что уже создано

- `.github/workflows` для CI и GitHub Pages.
- `src/data` для стартовых данных по картам, главам и ситуациям.
- `src/domain` для доменной модели, трактовок, прогресса и storage.
- `docs` для проектной документации.
- `reports` для отчётов по PR.
- `tasks/SU` для крупных задач.

## Цель PR-000

Собрать минимальный статический прототип браузерной игры «Путь Шута» на русском языке.

## Текущая игровая рамка

- Основной путь уже расширяется до всех 22 старших арканов.
- Младшие арканы будут добавлены позже как отдельный слой бытовых сцен и уточнений.
- UI остаётся мобильным, тёплым и статическим без backend.

## Объём PR-000

- Главный экран.
- Первый кликабельный сюжетный цикл.
- Сохранение прогресса в `localStorage`.
- Placeholder-экраны для остальных режимов.
- GitHub Actions для CI и Pages.

## Не входит в PR-000

- Полная колода из 78 карт.
- Сложная система раскладов.
- Backend и внешние API.
- Авторизация и серверное хранение.

## Базовые принципы

- Mobile-first.
- Статический фронтенд.
- Композиционная трактовка карты.
- Русский UI.
- Без слова «шпаргалка» в интерфейсе.
